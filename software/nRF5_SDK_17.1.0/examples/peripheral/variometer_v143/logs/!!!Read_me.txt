Чтобы конвертировать логи из Segger J0Link RTT Viewer в таблицу .csv, необходимо:

1. Предустановленный язык программирования Python v2
2. Исходный лог с именем .log в одной папке с программой log_to_csv.py
3. Совпадение имен исходного файла лога и открываемого файла в программе log_to_csv.py (по умолчанию - 1.log, программа редактируется: правая кнопка мыши - редактировать в IDLE)
